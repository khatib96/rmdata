# Company Messages Catalog (API Ready)

## الاسم
**أذكار وأدعية وعبارات تحفيزية لبرنامج الشركة**

## ملخص
هذا الملف يمثل نسخة **Production / API Ready** من المحتوى، ويتضمن:
- `meta`: معلومات تعريفية وإحصائية
- `selection_engine`: آلية الاختيار والأولوية
- `api_contract`: شكل مبدئي للتكامل مع الـ API
- `rules`: قواعد التفعيل الزمنية والسياقية
- `categories`: تعريف كل تصنيف
- `messages`: جميع الرسائل مع IDs ثابتة

## الإحصائيات
- إجمالي التصنيفات: 7
- إجمالي الرسائل: 100
- توزيع الرسائل:
  - morning: 16
  - evening: 11
  - motivation: 24
  - ramadan: 11
  - general: 23
  - quick: 10
  - friday: 5

## منطق التفعيل
1. `ramadan` أعلى أولوية عند تحقق شرط شهر رمضان هجرياً.
2. `friday` يفعّل طوال يوم الجمعة.
3. `morning` من 07:00 إلى 11:00.
4. `evening` من 16:00 إلى 18:00.
5. `motivation` يستخدم كـ fallback في الأوقات الأخرى.
6. `general` و `quick` متاحان دائماً كخيارات مساندة.

## أمثلة IDs
- `MSG-001`
- `MOR-001`
- `EVE-001`
- `MOT-001`

## ملاحظات تنفيذية
- جميع الرسائل `enabled: true` افتراضياً.
- جميع الرسائل باللغة العربية.
- تم اعتماد `Asia/Qatar` كتوقيت افتراضي للنظام.
- تم استخدام `weighted_random` كاستراتيجية اختيار افتراضية.
