# نظام إدارة الرداء الموحد - ERP System

نظام إدارة مكتبي متكامل مبني باستخدام Electron + React + TypeScript + TypeORM.

## 🚀 التقنيات المستخدمة

- **Frontend**: React 18 + TypeScript + Tailwind CSS
- **Backend**: Electron + TypeORM
- **Database**: SQLite (sqlite3)
- **State Management**: Zustand
- **Routing**: React Router
- **UI Components**: Lucide React Icons

## 📋 المميزات

### قاعدة البيانات
- ✅ نظام مستخدمين مع أدوار (SuperAdmin, Manager, Employee)
- ✅ إدارة الكيانات القانونية (Entities)
- ✅ إدارة الأفرع التشغيلية (Branches)
- ✅ إدارة الموظفين مع الربط الهجين (Legal Entity + Operational Branch)
- ✅ إدارة الأصول (المركبات والهواتف)
- ✅ إدارة السكن (Housing Units)
- ✅ نظام التنبيهات المركزي (Notifications)

### الواجهة
- ✅ لوحة تحكم رئيسية (Dashboard)
- ✅ شريط جانبي ديناميكي مع تصفية حسب الفرع/الكيان
- ✅ مركز تنبيهات مع ألوان (أحمر/أصفر)
- ✅ نظام تسجيل دخول مع صلاحيات
- ✅ تصميم عصري باستخدام Tailwind CSS

## 📦 التثبيت والتشغيل

### المتطلبات
- Node.js 18+ 
- npm أو yarn

### خطوات التثبيت

1. تثبيت الحزم:
```bash
npm install
```

2. تشغيل التطبيق في وضع التطوير:
```bash
npm run dev
```

3. بناء التطبيق للإنتاج:
```bash
npm run build
```

## 🗄️ قاعدة البيانات

قاعدة البيانات SQLite يتم إنشاؤها تلقائياً في مجلد `database/` عند أول تشغيل.

### الجداول الرئيسية:
- `users` - المستخدمون
- `entities` - الكيانات القانونية
- `branches` - الأفرع
- `employees` - الموظفون
- `vehicles` - المركبات
- `phones` - الهواتف
- `housing_units` - وحدات السكن
- `notifications` - التنبيهات
